# CodeAlpha Matrix Operations

## Description

This project implements Matrix Operations in C as part of the CodeAlpha C Programming Internship.

The program performs:

* Matrix Addition
* Matrix Multiplication
* Matrix Transpose

The implementation uses functions and two-dimensional arrays for better modularity.

## Features

* Matrix Addition
* Matrix Multiplication
* Matrix Transpose
* Function-based implementation
* Dynamic user input

## Technologies Used

* C Programming Language
* GCC Compiler
* Visual Studio Code

## How to Run

### Step 1: Clone Repository

git clone <repository-link>

### Step 2: Open Terminal

Navigate to project folder.

### Step 3: Compile

gcc matrix.c -o matrix.exe

### Step 4: Run

.\matrix.exe
